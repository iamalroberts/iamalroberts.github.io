PORTFOLIO

Stand

When I began taking violin lessons, I noticed that sometimes I would practice incorrectly all week only to realize it at my next lesson. If I could have shared my progress with my teacher throughout the week, any issues with my playing could have been caught early. This would lead to more effective practice and increase the speed by which I learned the instrument. I created Stand as an android app that allows the user to upload videos of themselves practicing for their teacher. The teacher can then leave comments as suggestions and tips as well as create tasks for the student to complete throughout the week. If a musician is not currently taking lessons, the app can still be used as a way to track progress over time.

Cognitive Distortion Journal

The basis of cognitive behavioral therapy (CBT) is identifying cognitive distortions (faulty or unhelpful ways of thinking) to effectively treat a number of issues ranging from depression and anxiety to addiction and eating disorders.﻿﻿﻿﻿﻿ These distorted thinking patterns cause negative feelings, which in turn can worsen an addiction. The Cognitive Distortion Journal is an Android mobile application that asks the user 5 questions to help focus their thinking and relive some of the stress that arises from cognitive distortions. It is built using the Google Sheets API as a backend to allow the user complete control over their entries. This way, the user has a private sheet in their Google Drive that they can access from any device, to do with as they see fit.
*Not to be used in place of actual mental health treatment from licensed professionals.

The Synesthesia Project

Synesthesia is a neurological condition in which information meant to stimulate one of your senses stimulates several of your senses. For those with synesthesia, sometimes words and numbers are associated with specific colors. I created the Synesthesia Project to allow users the ability to assign words their corresponding colors. This provides users with the satisfaction of seeing the words in print as they do in their mind. The website is built using the google sheets api to pull in the associations the user sets up in a google sheet. Future improvements would include a GUI that interfaces directly with reference data.
